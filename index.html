民生公共物聯網 API 資訊

本系統使用民生公共物聯網 (Civil IoT Taiwan) 的 OGC SensorThings API (STA) 公開資料集作為即時資料來源。

無需 API 金鑰！

該 API 屬於完全公開的環境數據，任何人皆可直接連線存取。系統已預設為您開啟即時串接功能，無需配置任何授權憑證或 Token 即可讀取全台最新雨量計數值。

API 連線詳情

  • 協議標準：OGC SensorThings API v1.0
  • 資料端點：`STA_RainSewer` 下水道雨量計
  • 提供單位：國家高速網路與計算中心 (NCHC)
  • 官方網址：sta.colife.org.tw
