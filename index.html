中央氣象署 API 設定

請輸入您的中央氣象署開放資料 API 授權碼。此金鑰僅會被儲存在您瀏覽器的本地快取 (LocalStorage) 中，不會傳送至任何第三方伺服器。

如何取得 API 授權碼？

  1. 前往 氣象署開放資料平臺網站
  2. 註冊並登入會員帳號
  3. 進入「個人資料」或「API 授權碼」頁面複製您的金鑰
