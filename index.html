NCDR API 設定

請輸入您的國家災害防救科技中心 (NCDR) 災防資料服務平台 API 授權金鑰 (Token)。金鑰僅會被儲存在您瀏覽器的本地快取 (LocalStorage) 中，不會傳送至其他第三方伺服器。

如何取得 NCDR API Token？

  1. 前往 NCDR 災害防救資料服務平台
  2. 註冊並登入會員帳號
  3. 申請您所需的降雨量 API 服務即可取得專屬 Token
